Lesson 1: Introduction to Python

Welcome to the first lesson on Python!

What is Python?

Python is a high-level programming language with dynamic semantics. It is known for its readability and simplicity, making it an ideal language for beginners.

Why Python?

  • Easy to learn: Python has a simple and straightforward syntax, making it easy for beginners to get started.
  • Powerful: It can handle complex projects, including data analysis and machine learning.
  • Flexible: Python can be used for prototyping, web development, and scientific research.

Your First Python Program

Let's start with writing the classic "Hello, World!" program. Open your code editor and type the following lines:

print("Hello, World!")

The print() function in Python outputs the specified text to the console. Run this code, and you'll see the greeting on the screen.

Python Syntax Basics

Python uses indentation to organize code. Here's an example of a simple loop:

for i in range(5):
    print("Number ", i)

This code prints numbers from 0 to 4, each on a new line.
