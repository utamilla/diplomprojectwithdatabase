Lesson 2: Data Types in Java

Explore the different data types and their uses in Java.

Understanding Data Types

Java is a strongly typed language, which means every variable and expression type is known at compile time. This strict type checking helps prevent errors and improves code reliability.

Data types in Java are divided into primitive types and non-primitive types. Primitive types are predefined by the language and named by a keyword, while non-primitive types are created by the programmer and can be used to call methods to perform certain operations.

Primitive Data Types

  • Integer: Represents whole numbers from -2,147,483,648 to 2,147,483,647 and used for integral values. Example: int age = 25;
  • Floating Point:Represents numbers with fractions, precisely up to 6 to 7 decimal digits. Example: float salary = 4832.15f;
  • Character: Represents symbols in a character set, like letters and punctuation marks. Example: char letter = 'A';
  • Boolean: Represents one bit of information either true or false. Example: boolean isJavaFun = true;

Non-Primitive Data Types

  • String: A sequence of characters used for storing text. Example: String name = "John";
  • Array: Stores multiple values of the same type in a single variable. Example: int[] array = {10, 20, 30, 40, 50};
  • Class: A blueprint for creating objects (a particular data structure), providing initial values for state (member variables or attributes), and implementations of behavior (member functions or methods). Example: public class Dog { String breed; int age; String color; }

Memory Management for Data Types

In Java, memory management is handled through automatic garbage collection, which helps in freeing up memory that is no longer needed. Primitive types are stored in the stack while non-primitive types are stored in the heap.
