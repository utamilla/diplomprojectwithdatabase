Lesson 1: Introduction to Java

Welcome to the first lesson on Java!

What is Java?

Java is a versatile and powerful programming language that follows the object-oriented paradigm and ensures platform independence through its Java Virtual Machine (JVM).

Why Java?

  • Robust: Well-suited for complex applications with its strong memory management and error handling capabilities.
  • Platform Independent: Java applications can run on any system that has the JVM installed, regardless of the underlying operating system.
  • Multi-threaded: Allows you to perform several tasks simultaneously within a program.

Your First Java Program

Let's write a simple program that prints "Hello, World!" to the console:

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

The `System.out.println()` method in Java is used to output text to the console. This method is part of the standard output class of Java's `System` library.

Java Syntax Basics

Unlike languages such as Python, Java does not use indentation to organize code; instead, Java uses braces `{}` to define the beginning and end of each function and control structure.
